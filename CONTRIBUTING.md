# Contributing

Wanna help? Cool.

## Setup

Fork the repo first, then:

```bash
git clone https://github.com/YOUR_USERNAME/framework.git
cd framework
composer install
```

## Before You PR

1. Run the tests: `composer test`
2. Fix the style: `composer cs:fix`
3. Make sure it actually works

That's it. No branch naming rules. No conventional commit sermons. Just don't break the build.

## Security

Don't open public issues for security stuff. See [SECURITY.md](SECURITY.md).
