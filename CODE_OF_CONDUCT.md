# Code of Conduct

Be cool. Be constructive.

If you like it here, contribute and help make Lightpack better.
If you don’t, feedback and criticism are always welcome — just keep things constructive.

Please don’t harass people, doxx anyone, or spam the community.

If someone is making the space unpleasant for others, please report it to pt21388@gmail.com.

Lightpack is MIT-licensed open source.
Use it, fork it, break it, improve it, and make it yours.
